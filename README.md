# 統一知識架構系統

本倉庫是一個跨領域的結構化知識系統，用於建構多層級的知識網絡模型。

## 涵蓋領域

- 數學系統（集合論、線性代數、機率論等）
- 物理系統（量子力學、熱力學、相對論等）
- 哲學系統（本體論、認識論、邏輯等）
- 經濟系統（賽局理論、制度經濟、行為經濟等）
- 控制論系統（回饋系統、自動控制、系統動力學）
- 演化系統（自然選擇、適應、文化演化）
- 資訊理論（熵、編碼、計算理論）

## 系統特徵

- 多層結構（基礎約束層 / 行為系統層 / 決策應用層）
- 節點化知識表示
- 跨領域關聯映射
- 可擴展知識網絡結構

## 系統目的

建立一個可用於理解複雜系統、AI建模與跨學科推理的統一框架。
